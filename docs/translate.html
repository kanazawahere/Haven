Propose your translation

One file per module you changed. For each:
  1. Copy the file contents below.
  2. Click Open in GitHub editor (sign in if prompted).
  3. Select all (Ctrl/⌘+A) and paste.
  4. Commit changes → Propose changes. GitHub forks the repo and opens a pull request for you.
