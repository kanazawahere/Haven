- Agent/MCP: send_terminal_input gained named keys (enter, esc, tab, ctrl-c/d/u/…, arrows, etc.) so an agent can send real control bytes — a clean Enter, Ctrl-C, line clears — that plain text couldn't express; an optional bracketed-paste mode for multi-line input into raw-mode REPLs (so it isn't folded into submit); and a returnSnapshot option that returns the resulting screen in the same call. Makes driving an interactive REPL over the agent endpoint far more reliable.
